II - About

Atelier Zero personal edition

An agent is only interesting when its loop can be inspected, corrected, and trusted.

Joseph works on the practical layer of AI systems: tools, memory, handoffs, verification, and the product surfaces that let humans stay in command. The work is less about making a model sound impressive and more about making an automated process legible.

This homepage treats that engineering stance like an editorial object: precise typography, collage imagery, and a restrained Rococo signal of gears, ornaments, and decision paths. The result is personal, but not ornamental for its own sake.

V - Method

Map the human workflow

Start from the current judgment process, not from the model feature list.

Step 02

Shape the agent contract

Define permissions, tools, stop conditions, and what the user must be able to inspect.

Step 03

Build the evaluation loop

Capture representative tasks, trace failures, and turn reviewer judgment into repeatable criteria.

Ship with escape hatches
